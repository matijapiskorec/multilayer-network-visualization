Server load button

If you want to offer a user the predefined locations from where he can load the network you can do it using the <server-load> directive:

<div ng-app="networkApp" ng-controller="networkCtrl">
    <server-load url="../data/trade-network-onelayer.json"></server-load>
    <server-load url="http://zel.irb.hr/matija/data?numberOfLinks=2000&products=[40,98,29,72]&countries=[59,53,37,12]&years=[1995,1996,1997,1998]"></server-load><br>
    <aspect-selector></aspect-selector>
    <force-layout></force-layout>
</div>
